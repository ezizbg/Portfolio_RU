## О проекте

Персональный сайт-портфолио для демонстрации навыков frontend-разработки, реализованных проектов и опыта.

Основной акцент сделан на адаптивной верстке, чистой структуре кода и использовании современных инструментов сборки.

---

## Стек технологий

* HTML5
* SCSS (Sass)
* JavaScript (Vanilla JS)
* Vite
* Prettier

---

## Функциональность

* Адаптивная верстка под различные устройства
* Структурированные стили с использованием SCSS
* Интерактивные элементы на JavaScript
* Сборка проекта с помощью Vite
* Раздел с проектами и контактной информацией

## Установка и запуск

Клонировать репозиторий:

```bash
git clone https://github.com/ezizbg/Portfolio_RU.git
```

Установить зависимости:

```bash
npm install
```

Запустить проект в режиме разработки:

```bash
npm run dev
```

---

## Сборка проекта

```bash
npm run build
```

Готовая версия проекта находится в папке `docs/`.
